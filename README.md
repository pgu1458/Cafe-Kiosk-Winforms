# 카페 키오스크 (Cafe Kiosk)

C# WinForms로 만든 카페 키오스크 시뮬레이터입니다.
실제 카페에서 쓸 법한 기능들을 직접 구현해봤습니다. 주문부터 결제, 회원 적립까지 한 흐름으로 동작합니다.

---

## 만든 이유

키오스크가 일상에서 워낙 자주 쓰이다 보니, 직접 만들어보고 싶었습니다.
UI 구성부터 결제 흐름, 다국어 처리, 시선 제어까지 최대한 실제 서비스처럼 구현하는 걸 목표로 했습니다.

---

## 주요 기능

### 시작 화면
- 한국어 / 영어 / 일본어 / 중국어 언어 선택
- 배경음악 재생 및 ON/OFF 토글
- 언어 선택에 따라 시작 문구 변경

### 주문 화면
- 음료, 디저트, 굿즈 탭으로 메뉴 구성
- 매장 / 포장 선택 (포장 시 10% 할인 자동 적용)
- 주문 목록 실시간 표시, 개별 취소 / 전체 취소
- 선택한 언어로 메뉴명 자동 번역
- 실시간 시계 표시

### 회원 기능
- 로그인 / 회원가입 (인메모리 가상 DB)
- 로그인 시 쿠폰 1개 자동 적립
- 쿠폰 10개 누적 시 무료 커피 1잔 제공 안내
- 회원 로그인 시 전체 금액 5% 할인 적용

### 결제
- 카카오페이 QR 결제
- 네이버페이 QR 결제
- 신용카드 결제
- 현금 결제 (카운터 안내)
- 결제 진행 중 로딩 애니메이션 표시
- 결제 완료 후 영수증 출력 선택

### 시선 제어 (Eye Tracking)
- OpenCV(OpenCvSharp) + Haar Cascade로 눈 위치 감지
- 시선이 특정 버튼에 일정 시간 머물면 자동 클릭
- ON/OFF 토글 버튼으로 활성화/비활성화

- .NET 6.0 (Windows)

## 사용 라이브러리

```
OpenCvSharp4          - 시선 제어용 카메라 입력 및 눈 인식
OpenCvSharp4.runtime  - OpenCV 런타임
AxWindowsMediaPlayer  - 배경음악 재생 (Windows Media Player COM)
```

## 실행 방법

1. 저장소 클론
```bash
git clone https://github.com/pgu1458/cafe-kiosk-winforms.git
```

2. Visual Studio에서 `Kiosk.csproj` 열기

3. NuGet 패키지 복원
```
OpenCvSharp4
OpenCvSharp4.runtime.win
```

4. `bin/Debug/net6.0-windows/` 경로에 아래 파일 확인
   - `click.wav`, `pay.wav`, `menu.wav`
   - `haarcascade_eye.xml`

5. 빌드 후 실행

## 한계 및 개선 방향

- 현재 사용자 데이터가 인메모리 딕셔너리라서 앱을 껐다 키면 초기화됩니다. DB 연동이 필요합니다.
- 시선 제어가 조명이나 카메라 환경에 따라 인식률 차이가 있습니다.
- 결제는 실제 API 연동 없이 UI 흐름만 구현된 시뮬레이션입니다.
